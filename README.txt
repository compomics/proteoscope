

   ==============
    Proteoscope
   ==============

   To start Proteoscope simply double click the jar file.

   From the command line use 'java -jar Proteoscope-X.Y.Z.jar'. 
   (Replace X.Y.Z with the Proteoscope version number.)

   For help with runnng Proteoscope please see:
   http://proteoscope.googlecode.com
